# Web-Development-ExamAethelguard Wiki is a web application that documents a custom Pathfinder 1e campaign world created by the Game Master.
The website provides lore, maps, characters, and interactive tools for both players and the master.
It is designed as a dynamic and responsive application built with HTML, CSS, and JavaScript, following the requirements of the Web Development final exam.
The application contains multiple sections that describe the world of Aethelguard and offer interactive features.
The goal is to provide a central hub where players can explore the setting, read about its history, view maps, and interact with various tools.
The project currently includes the following pages:

- Home, A general introduction to the world and the purpose of the wiki. At the bottom of the page, users can find a weather tool powered by the Open‑Meteo API, which allows them to input their real-world city and generate the corresponding weather in Aethelguard.
- Mappa, This section displays the world map and a written explanation of the continent’s geography. At the bottom, users can click on the names of cities to reveal their maps or a “coming soon” placeholder.
- Storia, A lore section structured like a history book, explaining the origins of the world, its regions, and the main religious elements.
- Personaggi, A collection of NPCs created for the campaign. New characters will be added over time as the story progresses.
- Contatto, A page where users can simulate login or registration. The interface allows switching between the two forms, and the submitted data is printed in the console to simulate backend behavior.

A top navigation bar is present on every page.
On smaller screens, it transforms into a hamburger menu for better usability.
Here the main features:

- Clickable city names that reveal maps
- Toggle buttons for login/register forms
- Dynamic weather display
- DOM manipulation across all pages

consistent style and design:

- A unified color palette
- Custom typography (Cinzel)
- Flexbox-based layout
- Media queries for mobile responsiveness

Future improvements
The project will be expanded with additional sections such as:

- Giocatori (Players) – Player character sheets
- Sessioni (Sessions) – Summaries of each game session

In conclusion Aethelguard Wiki is both a creative worldbuilding project and a functional web application.
It demonstrates interactive UI behavior, responsive design, DOM manipulation, and API integration, while presenting an original fantasy setting.
Enjoy exploring the world of Aethelguard!
